# 构建工具与配置

前端工程化必备的构建工具配置和优化技巧。

## 主流构建工具

### 📦 打包工具
- **Webpack** - 功能强大的模块打包器
- **Vite** - 新一代前端构建工具
- **Rollup** - ES Module 打包器
- **Parcel** - 零配置打包工具

### 🚀 开发工具
- **ESLint** - 代码质量检查
- **Prettier** - 代码格式化
- **Babel** - JavaScript 编译器
- **PostCSS** - CSS 转换工具

## 常见配置

- 多环境配置
