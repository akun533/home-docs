- **框架**: Vue.js、React、TypeScript
- **工程化**: Webpack、Vite、rollup
- **UI 库**: Element Plus、Ant Design、TailwindCSS

### 后端开发
- **语言**: Node.js、Java、Python
- **框架**: Express、Koa、Spring Boot
- **数据库**: MySQL、PostgreSQL、Redis、MongoDB
