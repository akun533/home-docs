# 如何使用 VuePress 博客功能

学习如何充分利用 VuePress 官方博客插件的强大功能。

<!-- more -->

## 简介

VuePress 官方博客插件 `@vuepress/plugin-blog` 为你的站点提供了完整的博客功能支持，包括：

- 📁 **分类管理**：按照不同的分类组织文章
- 🏷️ **标签系统**：为文章添加多个标签
- ⏰ **时间轴**：按时间顺序展示所有文章
- ✂️ **摘要生成**：自动或手动生成文章摘要

## Frontmatter 配置

每篇文章都可以在头部添加 YAML frontmatter 来配置文章信息：

```yaml
---
title: 文章标题
author: 作者名
date: 2025-12-28
categories:
  - 分类1
  - 分类2
tags:
  - 标签1
  - 标签2
  - 标签3
---
```

## 摘要分隔符

使用 `<!-- more -->` 作为摘要分隔符，它之前的内容会被作为文章摘要显示在列表页。

```markdown
文章开头部分会作为摘要显示...

<!-- more -->

这部分内容只在详情页显示。
```

## 功能特性

### 1. 分类页面

访问 `/category/` 可以看到所有分类，点击分类名称可以查看该分类下的所有文章。

### 2. 标签页面

访问 `/tag/` 可以看到所有标签云，点击标签可以筛选带有该标签的文章。

### 3. 时间轴

访问 `/timeline/` 可以按时间顺序浏览所有文章。

### 4. 文章列表

访问 `/article/` 可以看到所有文章的列表。

## 使用技巧

1. **合理使用分类**：建议使用 2-3 级的分类层级
2. **标签要精准**：每篇文章 3-5 个标签即可
3. **填写日期**：方便按时间排序和展示
4. **添加作者**：多人协作时很有用
5. **使用摘要**：让读者快速了解文章内容

## 总结

通过 VuePress 博客插件，你可以轻松打造一个功能完善的个人技术博客！
