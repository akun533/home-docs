## 为什么选择 VuePress？

VuePress 是一个基于 Vue 的静态网站生成器，它有以下优点：

- **简洁至上**：使用 Markdown 编写文档，专注于内容
- **Vue 驱动**：可以在 Markdown 中使用 Vue 组件
- **高性能**：预渲染生成静态 HTML，加载速度快
- **易于定制**：支持自定义主题和插件

## 开始写作

使用 VuePress 写博客非常简单：

1. 创建 Markdown 文件
2. 编写内容
3. 运行构建命令

```bash
# 开发模式
npm run docs:dev

# 构建生产版本
npm run docs:build
```
